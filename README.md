# Travel Booking Analytics Dashboard

## Project Overview
This project is an interactive Power BI dashboard built to analyze travel booking performance, revenue trends, booking channels, and destination insights.

The dashboard helps businesses understand customer booking behavior and identify top-performing travel destinations and channels.



## Dashboard Features

- KPI Cards for:
  - Total Revenue
  - Total Bookings
  - Average Booking Value

- Revenue Trend Analysis
- Booking Channel Distribution
- Top Destinations by Revenue
- Interactive Slicers and Filters
- Business Insights Panel



## Tools & Technologies Used

- Power BI
- Data Visualization
- Business Intelligence
- Data Analytics



## Key Insights

- Mobile App contributes highest bookings
- Revenue peaked during early booking months
- Singapore generated highest revenue
- Website and App dominate booking channels
